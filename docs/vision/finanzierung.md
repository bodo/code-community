# Finanzierung

## Grundsatz

Teilnehmer (Azubis, Umschüler) zahlen **nicht** für die Leistungen.

## Mittelbeschaffung

| Quelle | Beschreibung |
| --- | --- |
| **Umschulungsunternehmen** | Kooperationen mit Bildungsträgern |
| **Ausbildungsbetriebe** | Investieren in ihre Fachkräfte |
| **Spenden / Förderungen** | Unternehmen, Privatpersonen, IHK, Staat |
| **Alumni-Spenden** | Ehemalige Teilnehmer geben nach bestandener Prüfung etwas zurück |
| **Alumni als Mentoren** | Ehemalige Prüflinge unterstützen aktiv nachfolgende Jahrgänge |
| **JobCenter / Arbeitsamt** | Bildungsgutscheine nach AZAV-Zertifizierung (ab Q4 2026) |

## Mittelverwendung

| Bereich | Beschreibung |
| --- | --- |
| **AI Hardware** | GPUs, Server für self-hosted KI-Modelle und Lernumgebungen |
| **Lehrmittel & Lizenzen** | Fachliteratur, Prüfungsmaterialien, Software |
| **Ehrenamtspauschale** | Aufwandsentschädigung für Mentoren und Dozenten |
| **Infrastruktur** | Hosting, Netzwerk, Lernplattform, Chat |
| **Gehälter** | Später: Vollzeitkräfte bei ausreichender Finanzierung |

## Gemeinnützigkeit

- Förderung der Berufsbildung
- Keine Gewinnausschüttung
- Spendenquittungen möglich
- Transparente Finanzierung

## Crowdfunding

Es wird eine eigene Non-Profit Crowdfunding-Plattform entwickelt (siehe [technik/crowdfunding.md](../technik/crowdfunding.md)), um Kampagnen für spezifische Bedarfe durchzuführen.
