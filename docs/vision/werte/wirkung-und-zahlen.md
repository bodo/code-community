## SROI: Social Return on Investment

**Konservativ 8:1** — jeder investierte Euro spart der Gesellschaft 8 € an Folgekosten.

| Szenario | SROI |
| --- | --- |
| Brutto (ohne Korrekturen) | 16:1 bis 122:1 |
| **Konservativ (mit 30 % Deadweight, 20 % Attribution)** | **8:1** |
| Worst-Case (bei nur 70 % Bestehensquote statt >90 %) | 5:1 |

### Kostenvergleich

| Szenario | Kosten |
| --- | --- |
| **Durchfallen / Abbruch** | 29.000–36.500 € (Bürgergeld, erneutes Schulgeld, entgangene Steuer) |
| **Gezielte Prüfungsvorbereitung bei uns** | 1.800–3.600 € pro Teilnehmer |
| **Vermiedene Folgekosten pro Erfolg** | 25.000–33.000 € |

**Quelle:** Detailrechnung in `bpw/businessplan.typ` Kapitel 7.5 "SROI-Herleitung".

## Ökologische Indikatoren

| Maßnahme | Effekt |
| --- | --- |
| **100 Refurbished-Laptops** (perspektivisch) | 20–40 Tonnen CO2 vermieden gegenüber Neuproduktion |
| **Self-hosted statt Cloud-Hyperscaler** | Eigene Server in Deutschland, mit Ökostrom-Anforderung an Hosting-Partner |
| **Open-Source-Software** | Kein proprietärer Overhead, reproduzierbar, keine Lock-ins |
| **Papierlose Lehrmittel** | Digitale Prüfungssimulationen, Markdown-basierte Dokumentation |
| **Remote-First** | Reduzierung von Pendelwegen und CO2-Emissionen durch Fahrt zu Präsenzterminen |

**Ökostrom-Nachweis beim aktuellen Hosting-Partner** ist eingeplant und wird im nächsten Wirkungsbericht ausgewiesen.

## Methodik

- **Wirkungsbericht nach Social Reporting Standard (SRS)** — etablierter Rahmen für gemeinnützige Social Enterprises, entwickelt von Phineo, Ashoka, Bertelsmann u.a.
- **Input → Output → Outcome → Impact** als Wirkungskette, nach SRS strukturiert
- **Transparenz vor Präzision** — wir publizieren auch unvollständige Berichte. Besser eine ehrliche Schätzung mit dokumentierten Unsicherheiten als ein perfektionierter Report, der nur positive Zahlen zeigt.
