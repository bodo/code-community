# Außenkommunikation -- Übersicht

## Kommunikationsstrategie

Die Außenkommunikation verfolgt zwei parallele Ziele:

1. **Teilnehmer gewinnen** -- Azubis, Umschüler und Extern-Prüflinge auf das Angebot aufmerksam machen
2. **Unterstützer mobilisieren** -- Betriebe, Sponsoren, Ehrenamtliche und die öffentliche Hand überzeugen
