# Lehrplan & Angebot

## Kernleistungen

### Nachhilfe

Gezielte Vorbereitung auf die schriftlichen IHK-Prüfungen (AP1 und AP2) für Fachinformatiker Anwendungsentwicklung und Systemintegration. Durcharbeitung typischer Prüfungsfragen, Diagramme und Musteraufgaben.

### Prüfungsvorbereitung

Intensivtraining in Kleingruppen in den 4 Prüfungsphasen des Jahres (Februar, April, August/September, November). Fokus auf prüfungsrelevante Themen, nicht auf breites Curriculum.

### Coaching & Mentoring

Persönliche Begleitung durch erfahrene IT-Fachkräfte über die gesamte Vorbereitungszeit. 1-zu-1 Mentoring und Gruppenbetreuung. Details zum Mentoring-Format: [Zielgruppenansprache → Mentoren](zielgruppen-ansprache.md#ehrenamtliche-mentoren)

### Projektbetreuung

Unterstützung beim IHK-Abschlussprojekt: Themenfindung, Planung, Umsetzung, Dokumentation und Präsentationsvorbereitung.

## Formate

### Intensivkurs AP1/AP2

| Aspekt | Details |
| --- | --- |
| **Dauer** | 3-4 Wochen, 2-3 Abende/Woche à 2h |
| **Zeitpunkt** | 6-8 Wochen vor jeder IHK-Prüfung (4x/Jahr) |
| **Gruppengrösse** | 5-15 Teilnehmer |
| **Format** | Präsenz oder Remote |
| **Inhalt** | Typische Prüfungsaufgaben durcharbeiten, Wissenslücken schliessen, Prüfungsstrategie |
| **Durchführung** | Ehrenamtliche Dozenten mit vorbereiteten Materialien |

### Regelmässige Nachhilfe

| Aspekt | Details |
| --- | --- |
| **Dauer** | 1-2h/Woche, laufend |
| **Format** | Remote (Kleingruppe oder 1:1) |
| **Inhalt** | Ein Fachgebiet pro Session (z.B. Netzwerktechnik, SQL, Projektmanagement) |
| **Durchführung** | Ehrenamtliche Dozenten, abends |

### Probeworkshop (kostenlos, offen)

| Aspekt | Details |
| --- | --- |
| **Dauer** | 2h, einmalig |
| **Zeitpunkt** | 8-10 Wochen vor Prüfung |
| **Zweck** | Appetizer: Format, Dozenten, Methodik kennenlernen. Keine Anmeldung nötig. |
| **Ziel** | Teilnehmergewinnung durch Erleben statt Versprechen |

### Spezial-Workshop

| Aspekt | Details |
| --- | --- |
| **Dauer** | 2-3h, einmalig |
| **Themen** | z.B. "UML-Diagramme verstehen", "Netzwerkplanung für die AP2", "Wirtschaftlichkeitsberechnung Schritt für Schritt" |
| **Format** | Präsenz oder Remote |
| **Durchführung** | Dozent mit vorbereiteten Materialien, eigenständig planbar |

### Coding-Dojo

| Aspekt | Details |
| --- | --- |
| **Dauer** | 2h, unregelmässig |
| **Inhalt** | Programmieraufgaben im Prüfungsformat, Pair Programming, Live-Coding, gegenseitiges Review |
| **Zielgruppe** | Schwerpunkt AE-Prüflinge |
| **Format** | Präsenz oder Remote |

### 1:1 Mentoring ("Daily")

| Aspekt | Details |
| --- | --- |
| **Dauer** | 15 min/Tag, über Wochen/Monate |
| **Format** | Chat oder Video-Call |
| **Verhältnis** | 1 Mentor : 1-2 Prüflinge |
| **Fokus** | Motivation, Verbindlichkeit, persönliche Hindernisse -- nicht Fachwissen |
| **Durchführung** | Ehrenamtliche Mentoren (Senior-Devs, Lehrer, Ruheständler) |

### IHK-Abschlussprojekt-Betreuung

| Aspekt | Details |
| --- | --- |
| **Dauer** | 4-6 Wochen, 2-3h/Woche |
| **Inhalt** | Thema eingrenzen, Zeitplan erstellen, Dokumentation reviewen, Präsentation proben |
| **Verhältnis** | 1 Dozent : 1-3 Prüflinge |
| **Voraussetzung** | Dozent mit eigener Projekterfahrung |

### Prüfungssimulation

| Aspekt | Details |
| --- | --- |
| **Dauer** | 4h (Klausur + Nachbesprechung) |
| **Zeitpunkt** | 2-3 Wochen vor Prüfung |
| **Inhalt** | Realistische Probeklausur im IHK-Format unter Zeitdruck. Individuelle Korrektur und Rückmeldung. |
| **Häufigkeit** | 1x pro Prüfungsphase |

## Lernmaterialien -- Produktion und Kuratierung

### Eigene Materialien (Lehrmittel-Autoren)

| Materialtyp | Beschreibung | Plattform |
| --- | --- | --- |
| **WISO-Fragepool** | Multiple-Choice-Fragen, regelmässig gegen aktuelle Gesetzeslage geprüft | wiso.abschluss.jetzt |
| **AP1/AP2-Trainingsfragen** | Aufgaben im IHK-Format mit Musterlösungen | Lernplattform (bodo-os) |
| **Cheatsheets** | 1-2 Seiten pro Thema (OSI-Modell, UML, SQL, Subnetting). PDF, druckbar, CC-lizenziert. | Nextcloud + Website |
| **Prüfungssimulationen** | Realistische Probeklausuren mit Zeitvorgabe und Auswertung | Lernplattform |
| **Erklär-Diagramme** | Mermaid/SVG-Diagramme zu Kernthemen | BookStack + Lernplattform |

### Kuratierte Lernpfade (Lehrmittel-Autoren)

Für jedes Themengebiet werden die besten frei verfügbaren Ressourcen gesammelt und in einen strukturierten Lernpfad geformt:

| Quelle | Einsatz |
| --- | --- |
| **YouTube-Serien** | Kostenlose Kurse zu Netzwerktechnik, Programmierung, Datenbanken. Geprüft auf Aktualität und Qualität. |
| **Udemy-Kurse** (5-15 EUR) | Empfehlungen für vertiefende Kurse, wo kostenlose Quellen nicht ausreichen |
| **Dokumentationen** | Offizielle Docs (z.B. PostgreSQL, Python, RFC), Tutorials |
| **Interaktive Übungen** | Externe Plattformen (z.B. SQL-Trainer, Subnetting-Rechner) |

Die Lernpfade werden auf der Lernplattform (my-paths) bereitgestellt -- hauptsächlich zum Selbststudium, ergänzt durch Dozenten-geführte Sessions.

### Content-Gap-Analyse

Beim Kuratieren wird systematisch festgehalten, wo gutes Material fehlt:

- Thema nur auf Englisch oder veraltet verfügbar
- Kein prüfungsspezifisches Material vorhanden (IHK-Format)
- Bestehende Ressourcen nicht auf deutsches Berufsbildungssystem zugeschnitten

Diese Lücken werden priorisiert und als Eigenproduktion geplant (Erklär-Videos, Screencasts, Cheatsheets).
