# Website & Präsentationen

## Website: abschluss.jetzt

Die öffentliche Website ist ein Astro-basierter One-Pager unter der Domain `abschluss.jetzt`. Eine zweite Domain `abschluss.coach` dient als Sales Page.

| Domain | Zweck | Technologie |
| --- | --- | --- |
| abschluss.jetzt | Landing Page | Astro (PHP on VPS) |
| abschluss.coach | Sales Page | PHP on VPS |
| help.abschluss.jetzt | Volunteer-Info | Static Site |

**Projekt-Verzeichnis:** `../../website/`

## Präsentation

Eine interaktive Reveal.js-Präsentation stellt das Projekt in einer Hackerspace-Community vor. Sie enthält:

- Allgemeine Projektvorstellung (Mission, Problem, Vision, Timeline, Finanzierung)
- 9 rollenspezifische Detailbereiche (Entwickler, SysOp, Mentoren, QM, Marketing, Teilnehmer, Sponsoren, Betriebe, Öffentliche Hand)
- Multiplex-Funktion (Live-Steuerung bei Vorträgen)
- Matomo-Tracking integriert

**Datei:** `../../organization/presentation/index.html`

## Raumnutzung VR-Nutzergemeinschaft

Die Prüfungsvorbereitung soll teilweise in Räumen eines Hackerspaces stattfinden:

- 4 Perioden pro Jahr, jeweils 3--4 Wochen
- Tagsüber 15--19 Uhr
- Intensivtraining in Kleingruppen, ruhiges fokussiertes Arbeiten
- Co-Existenz mit dem regulären Hackerspace-Betrieb
